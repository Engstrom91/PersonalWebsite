Doctor Review API

About the project

I wanted to get better at solving problems for actual organizations so I developed an application that would let patients in a hospital rate their doctors. The managers at the hospital could then look at the feedback and give suggestions to the doctors for how to become more service minded.

This CRUD application stores the doctor objects in a H2 database and they are accessed via JPA. I followed the MVC design pattern to build this application.
